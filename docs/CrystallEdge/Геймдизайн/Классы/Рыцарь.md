# Рыцарь

Базовый класс, доступный для новичков, с простыми механиками.

Роль: Танк

Стилистика: Грубость, красно-коричневые тона, ближний бой, сталь.

 ## Классовые механики

### 1. Сила
Увеличение исходящего физического урона в ближнем бою.

### 2. Временные щиты
Дополнительные щиты поверх здоровья, блокирующие урон определенного типа.
