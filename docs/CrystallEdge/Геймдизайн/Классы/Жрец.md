# Жрец

Базовый класс, доступный для новичков, с простыми механиками.

Роль: Поддержка

Стилистика: бело-голубые тона, "святость", священный белый огонь, чистота.

## Классовые механики

### 1. Лечение
Восстановление здоровья себе и другим существам.

### 2. Божественный щит
Блокирует любое количество урона за единицу божественного щита.
