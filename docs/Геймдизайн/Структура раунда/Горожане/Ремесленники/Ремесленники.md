# Ремесленники

Подгруппа горожан, помогающая страже вычислить антагонистов побочными методами.

Может быть большое количество различных ремесленников, каждый из которых помогает разными способами.

## Цели

1) Выжить до окончания раунда.
2) Опционально, каждый горожанин может иметь до 3 ролейплей целей-амбиций.
