# История разрушающегося песочного замка
В самом концепции ss14 написано "История станции на которой все идет не так".

Каждый раунд начинается с работающей станции, и случайные события, антаги, чел. фактор ломает эту станцию, создавая интересные истории. 

Как только весь потенциал разрушения песочного замка кончается, играть обычно становится скучно.
