# Трактир
Трактир это фракция, занимающаяся заработком денег за счет предосталвения сервиса для других игроков. Еда, напитки, ночлег.

Оперирует следующими ресурсами:
- Сырая еда
- Готовая еда
- Напитки
- Деньги

## Краны
Роль фермера, работающая с разными видами огородов и источников ресурсов:
- Огород
- Грибная ферма
- Пчелиная пасека

Фермеры добывают различные ресурсы для трактира

## Инвентарь
В таверне есть склад + инвентарь каждого игрока из фракции таверны
