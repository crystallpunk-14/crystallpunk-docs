# Магоинженерия (Artifice)
Технологический прогресс Эберрона можно сравнить с Землей 

Существует 3 вида осколков дракона, энергия из которых используется в разных целях:
- Осколки Сибериса. Падают с кольца Сибериса. Усиливают магические способности меток после фокусировки. Используются для создания артефактов, которые могут использовать носители меток.
- Осколки Эберрон. Находятся в жеодах под землей. Могут хранить в себе заклинания, или использовать их самостоятельно. Используются как книги заклинаний для псиоников, или для артефактов с пассивными эффектами (фонари, холодильники)
- Осколки Хайбер. Находятся глубоко под землей, возле магмы. Используются гномами Зиларго для связывания элементалей, и использования их элементальной энергии в сложных механизмах (корабли, поезда, т.д.)

Эти осколки не являются единственными инструментами для работы с энергиями, они модификаторы, усилители базовой системы, и при этом каноничная часть сеттинга.

Артефакторы - распространенная профессия, они используют Тайную магию для создания
